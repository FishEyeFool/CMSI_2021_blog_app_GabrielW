# Gabriel's Blog

Gabriel's Blog is a simple blog application built with React and Firebase. It allows users to sign in, create, view, and delete articles.

## Features

- User authentication (sign in and sign out)
- Create new articles
- View a list of articles
- Delete articles

- React
- Firebase Firestore
